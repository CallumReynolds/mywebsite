Matching Pairs Game

During my mid semester TAFE break, I decided to partake in the The Complete 2019 Web Development Bootcamp by Angela Yu on Udemy which I have since completed. While progressing through the course, my TAFE teachers sent out an email with an optional task to complete, which involved: refactoring an incomplete matching pair game in Java. Using my initiative I decided to develop my own Matching Pairs Game using my newly acquired skills. I have the game source code on my github and hosted on github pages.

Simple Jekyll Page

I discovered Jekyll while looking at hosting websites. The idea of being able to host a website with data without requiring a database sounded like a niche way of developing a site. Using a Windows computer made it a little bit of a difficult process because Jekyll requires Ruby which isn't supported well on Windows. I was able to get the very basics of the Jekyll site up and running and can be previewed here, with some recipes you may like!

TAFE Projects

These are a sample of the applications that I have created for TAFE assessments:

  • Tool Rental Web App - ASP.NET MVC web application which communicates with a RESTful API to enable the tracking and rentals of tools. Features include systems to manage customers, tools and rentals(hiring and returning) tools. Also including reports and history of transactions which can be exported to a csv file.

  • 'Quick Quiz' - Java inter-processing communication system that delivers quiz questions via a classroom-based network to a group of students. Using a chat server which the teacher and students connect to, the teacher has a seperate form to the students. The teacher has a list of questions to choose from and send to the students. Once completed the students sends answers back for the teacher to review. Data is controlled using doubly linked lists and binary trees.

  • 'Xeno Onslaught' - a mobile game built using the LOVE 2d engine which can handle game states. A simple game which involves a player facing off an endless wave of aliens and tries to survive for as long as possible. Has settings to change game options and a scoreboard which keeps track of player scores.

TAFE Projects

  • Tool Rental Windows App - very similar functionality to the Tool Rental Web App as they are both based off the same scenario. Created as a windows form application using SQLite and Dapper for data handling.

  • 'Ian's Industrial Installation' - java form application with the main focus of handling 2d arrays. It worked by reading in CSV files with formatted data and displaying the corresponding data in grid view. Different files could be read in and exported out as well.

  • 'Birthday Tracker' - a simple java form application which tracks certain information About people such as birthday, likes and dislikes. Used simple data structures and had features for sorting and navigation.
